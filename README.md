# To-Do List Project

Aplicação web simples para gerenciamento de tarefas desenvolvida com **HTML, CSS e JavaScript puro (Vanilla JS)**.

O usuário pode adicionar tarefas, marcar como concluídas e excluir itens da lista. As tarefas são armazenadas no **Local Storage**, permitindo que permaneçam salvas mesmo após atualizar ou fechar o navegador.

🔗 **Acesse o projeto online:**  
https://cristina-freitas.github.io/To-Do-List-Project/

---

# 📌 Sobre o Projeto

Este projeto foi desenvolvido como prática de **manipulação do DOM, eventos e armazenamento de dados no navegador**, utilizando apenas tecnologias básicas do front-end.

A aplicação permite ao usuário:

- Adicionar novas tarefas
- Marcar tarefas como concluídas
- Excluir tarefas da lista
- Visualizar a quantidade total de tarefas
- Visualizar a quantidade de tarefas excluídas
- Manter os dados salvos no navegador através do **Local Storage**
